Research-Grade Wearable Alpine Ski Posture Monitoring System
🏛 HUST · Nov 2024 – Present · Actively In Progress
Status: Hardware prototyping and algorithm development ongoing.

System Overview
7-node distributed IMU system (ESP32 + MPU-9250) for high-precision 3D lower-limb motion capture during alpine skiing. Nodes: waist (master), bilateral shanks, ankles, ski boards.

Key Measurements
• True edge angle — waist IMU low-pass filter removes slope tilt; targets <5° L/R asymmetry for elite carving
• Knee flexion — quaternion relative rotation waist→shank
• Ankle dorsiflexion — shank→ankle joint kinematics (boot forward lean)
• Fall detection — any node acceleration >3 g triggers alarm

Hardware Architecture
1× ESP32 master (waist) handles data aggregation, WiFi/BLE, SD card. 2× ESP32-C3 sub-nodes (left/right leg) each manage 3× MPU-9250 via TCA9548A I²C multiplexer at 200 Hz. ESP-NOW wireless sync achieves <1 ms timestamp alignment. Total BOM ~¥500–600.

Algorithm
Madgwick AHRS in 6-axis mode (magnetometer disabled: ski binding ferromagnetic interference). Dynamic β switching (0.033 low / 0.1 high dynamic). All joint angles computed via quaternion relative rotation to avoid gimbal lock.
